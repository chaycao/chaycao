## Open Source Contribution

### [oddeNova](https://github.com/yiichuan/oddeNova)

Contributor to oddeNova, an AI-powered live music creation platform in the browser.

oddeNova lets creators describe a mood, theme, or musical idea in natural language, then uses an AI Agent to generate and iterate Strudel live coding so the music can be played instantly in the browser.

What interests me about this project:
- AI-assisted music creation
- Human-in-the-loop creative workflows
- Agentic generation and iteration
- Browser-based music tools
- Practical applications of Vibe Coding

Tech focus: React, TypeScript, Vite, Strudel, WebAudio, AI Agent workflows
