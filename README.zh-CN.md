## 开源贡献

### [oddeNova](https://github.com/yiichuan/oddeNova)

我是 oddeNova 的贡献者之一。oddeNova 是一个运行在浏览器中的 AI 驱动实时音乐创作平台。

创作者可以用自然语言描述一种情绪、主题或音乐想法，oddeNova 会通过 AI Agent 生成并迭代 Strudel live coding，让音乐可以直接在浏览器中播放。

这个项目吸引我的地方：
- AI 辅助音乐创作
- Human-in-the-loop 创作工作流
- Agentic 生成与迭代
- 浏览器中的音乐工具
- Vibe Coding 的实际应用

技术关注：React、TypeScript、Vite、Strudel、WebAudio、AI Agent 工作流
